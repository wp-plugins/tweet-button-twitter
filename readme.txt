This FREE plugin allows you to easily add the Twitter Button anywhere in your wordpress blog.

== Description ==

Tweet Button for Twitter allows you to easily add the tweet button into every post and page of your website.

* Choose where to display the tweet button (Posts,Pages,Everywhere).
* Select the button layout (Vertical,Horizontal,No Count)
* Select the button placement (Top Left,Top Right,Bottom Left,Bottom Right .. etc)
* Set the margins (Top,Bottom,Left and Right) between the Twitter Button and your surrounding content
* Select a twitter account (if any) which you want to be mentioned in each tweet (e.g. via @mytwitteracc)

== Installation ==

1. Upload the folder `tweet-button-twitter` folder to your `/wp-content/plugins/` directory
2. Activate `Tweet Button Twitter` through the 'Plugins' menu in WordPress
3. Go to 'Settings' -> `Tweet Button Twitter` to customize how your tweet button(s) should be displayed
